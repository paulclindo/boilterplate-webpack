# My personal React Boilerplate

My personal Boilerplate for react project using webpack, react and styled components

### Prerequisites

Install

```
npm install
```

### Installing

For development env running

```
npm run dev
```
For production env

```
npm run build
```

## Running the tests

```
npm run test
```

## Built With

React, webpack, Graphql & Jest
